# 学习笔记 | Learning Notes

## 项目概览 | Project Overview

这个项目让我第一次接触完整的工具开发流程。

## Lesson 1 — 写代码只是开发的一部分

真正的开发还包括：

- 环境配置
- 路径管理
- 工具集成
- 调试

## Lesson 2 — 理解 PATH

安装软件不等于终端能够找到软件。

## Lesson 3 — PowerShell 与 CMD 的区别

不同终端环境可能导致不同结果。

## Lesson 4 — 文件路径的重要性

路径问题往往比 Python 语法问题更常见。

## Lesson 5 — bat 文件的作用

bat 文件可以把脚本包装成更易用的命令。

## Lesson 6 — Debug 是开发的一部分

调试不是失败，而是开发过程的一部分。

## Lesson 7 — AI 辅助开发

AI 提供建议，但最终仍然需要自己验证和解决问题。

## 最大收获 | Biggest Takeaway

我第一次完整经历了：

需求 → 开发 → 调试 → 工具落地

这一完整过程。
